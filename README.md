# Employee Management System

A Spring Boot MVC application for managing employee records.

## Technologies Used

- Java 17
- Spring Boot
- Spring Data JPA
- Thymeleaf
- MySQL
- Bootstrap 5
- Maven

## Features

- Employee Registration
- Save Employee Data
- Responsive UI
- MySQL Database Integration

## Future Enhancements

- Employee List
- Update Employee
- Delete Employee
- Search Employee
